# README

## Main URIs
This repository describe the content of the HAMLET ontology, which is an extension of the PeGazUs ontology.
The TABULAE ontology is dedicated to the representation of land plots evolution using assertions extracted from (historical) documents.
It's URI is
```
http://rdf.geohistoricaldata.org/def/cadastre#
```

The following URI is the one of the core ontology developped by (Bernard et al.)
```
http://rdf.geohistoricaldata.org/def/address#
```

## Codes URIs
Some of the SKOS concept have French names because they have been specificaly developped using the French Land Registry (both historical and modern). When they have equivalent names in english, they are provided using skos:PrefLabel@en.
